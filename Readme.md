# Personal Ledger

---

A notepad based on React Hooks + Vite 2.0 + Zarm UI

## Technology stack

---

- Egg.js
- Node.js
- React
- React Hook
- MySQL
- Vite

## Getting Start

---

### Installation & run

Installing dependencies:

`npm install `

Run code:

`npm run dev`
